LEVEL 2

RESTRICTIONS:

Only use HTML
In addition to the elements allowed in Level 1, you can use any of the following HTML elements:

  • div (block element)
  • span (inline element)
  • h1, h2, h3, h4, h5, h6 (headings)
  • strong (strong text)
  • em (emphasized text)
  • ol, ul, li (lists)
  • a (links: do not link to external sites, but you can link to parts of your poem using an attribute to assign an id)
Again, you can use &nbsp; to equal a visible space.

GOALS:

Focus on presenting the text in different hierarchies and scales using only HTML.
